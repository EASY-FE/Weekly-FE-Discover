## react教程 prd

### 1. 本教程面向人群

零基础，或有vue开发经验但未接触过react的人。以白话、简易、易懂的形式制作一系列react的入门教程，保证阅读人员看完之后可以直接上手开发react项目，并可以带着理解去阅读官方文档。

### 2. 教程关注点

对于初学者来说，刚接触react的话大致有以下痛点问题

+ 如何理解JSX语法
+ react的数据流，如何理解state、prop
+ virtual dom
+ 如何理解react组件化编程的思想
+ 如何通俗的讲好redux
